Kurz-Anleitung

1. Gerät verbinden

ESP32-C6 via USB anschließen. Unterstützte Browser: Chrome, Edge, Opera, Arc (WebSerial).

2. BIN-URL einfügen

Öffentliche URL zur .bin eintragen (z. B. GitHub-Release, CDN, eigener Host).

3. Manifest erzeugen

Ein Klick auf Manifest erstellen generiert ein temporäres ESP-Web-Tools-Manifest aus der URL.

4. Flashen

Mit dem erscheinenden Button wird die Verbindung aufgebaut und die Firmware geschrieben.

Hinweise

  • • Geräte ggf. mit BOOT/EN in Flash-Modus setzen (je nach Board).
  • • Manifest nutzt Offset 0x10000 (65536) für App-BIN – wie üblich für ESP-Images.
  • • Erase-Prompt ist aktiv: Neuinstallation löscht vorherige Firmware-Daten.
  • • Alles läuft lokal im Browser; keine BIN wird hochgeladen.
